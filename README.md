## 🚀 Projects

### 🛡️ Journey Shield – Travel Safety App
🔗 [Live Demo](https://journey-shield.vercel.app)

A comprehensive travel safety application integrating real-time safety assessments, dynamic routing, and location discovery via third-party APIs.
- Built a robust Node.js/Express backend with secure JWT authentication and Role-Based Access Control (RBAC) to manage distinct Traveler and Guide dashboards.
- Designed a responsive React frontend featuring interactive Leaflet maps, synthesizing complex multi-source data into actionable user insights.

**Tech Stack:** MERN Stack · TomTom API · OpenWeatherMap API · Ticketmaster API · JWT · RBAC · Leaflet Maps

---

### 💬 TrueWAY Messenger – Privacy-First Social Platform

A privacy-centric social platform enforcing strict anonymity with an "avatar-only" interface and ephemeral data retention.
- Implemented MVVM architecture with Firebase Firestore for real-time updates and a "zero-knowledge" user privacy protocol.
- Integrated AI-driven content moderation to verify news validity and auto-purge toxic content.

**Tech Stack:** Android · Java · Firebase Firestore · MVVM · GenAI

---

### 🤝 Real-Time Collaboration Platform

A full-stack communication suite featuring instant messaging, a synchronized collaborative whiteboard, and video calling.
- Integrated Agora.io SDK for high-quality video calling with a custom draggable Picture-in-Picture (PiP) interface.
- Engineered a secure backend with MongoDB for persistent message storage and JWT for session management.

**Tech Stack:** MERN Stack · Socket.IO · WebRTC · Agora.io SDK · MongoDB · JWT

---

### 🔍 List Lens – Image Scanner (OCR Utility)
🔗 [Live Demo](https://list-lens-app.vercel.app)

A productivity app that converts image data into editable, actionable checklists using Computer Vision, with secure auth and cross-device sync.
- Integrated a Computer Vision (OCR) pipeline using Tesseract.js to transform unstructured image data into structured checklists.
- Engineered data isolation logic for user-specific file access with a mobile-responsive UI and real-time auto-sorting.

**Tech Stack:** MERN Stack · Tesseract.js · MongoDB Atlas · Computer Vision
